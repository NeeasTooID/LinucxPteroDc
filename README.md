# Pterodactyl Bot

Pterodactyl Bot is a Discord Bot that controls [pterodactyl](https://pterodactyl.io) panel

## Installation

Usage The Command Line to Download And install modules 

```
git clone https://github.com/NeeasTooID/LinucxPteroDc.git
```
# Install Module
```
npm install
```
